Camera Configuration

Camera configuration is stored in cameras.json file. To modify cameras, edit this file directly. 

{
"camera1": {
"name": "Camera Name",
"rtsp_url": "rtsp://user:pass@ip:554/stream",
"latitude": 40.7128,
"longitude": -74.0060,
"auto_recording": true
}
}

Add `latitude`, `longitude`, and `auto_recording` (true/false) to enable automatic recording between local sunset and sunrise.
